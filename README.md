# InfraOps API Platform

A simple REST API built with FastAPI for managing infrastructure-related services.

## Features
- Get all services
- Create a service
- Swagger documentation

## Tech Stack
- Python
- FastAPI
- Uvicorn

## Run locally
```bash
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m uvicorn main:app --reload
=======
